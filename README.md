## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Gen File
```bash
#gen controller
$ nest g co modules/{file_name} --no-spec

#gen module
$ nest g module modules/{file_name}

#gen service
$ nest g s modules/{file_name} --no-spec
```
